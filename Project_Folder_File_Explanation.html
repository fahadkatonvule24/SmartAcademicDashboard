IUIU Kampala Campus Academic ERP System

Project folder and file explanation - generated June 15, 2026

This project is a FastAPI-based academic ERP dashboard. The main deployed app starts from main.py, which exposes iuiu_dashboard.api:app. The app combines an ERP dashboard, NLP translation support, and a course assistant that indexes lecture resources.

High Level Flow

LayerWhat It Does
main.py Imports the integrated FastAPI app from iuiu_dashboard.api. Hosting platforms use this as the entry point.
iuiu_dashboard/api.py Defines HTTP routes for login, workspaces, resources, quizzes, feedback, PDFs, translation, chat, and admin tools.
iuiu_dashboard/service.py Contains the main business logic and coordinates the translation and course assistant modules.
data/ Stores local JSON data and uploaded resources. It acts as the project's lightweight database.
static_dashboard/ Provides the main browser interface used by students and lecturers.
